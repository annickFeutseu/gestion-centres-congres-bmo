## Prérequis

- **Docker** et **Docker Compose** installés

> **Machines ISTIC** : activer Docker d'abord avec `dockerd-rootless-setuptool.sh`

---

## Installation et lancement

```bash
# 1. Cloner le dépôt
git clone <url-du-depot>
cd <nom-du-dossier-projet>

# 2. Lancer l'application
Lancer docker desktop puis taper la commande ci-dessous:
docker compose up
```

La première exécution télécharge les images Docker — prévoir quelques minutes.

---

## Accéder à l'application

Une fois les conteneurs démarrés, ouvrir dans un navigateur :

```
http://localhost:3000/
```

---

## Arrêter l'application

```bash
# Arrêter
docker compose down

# Arrêter et supprimer les données
docker compose down -v
```
